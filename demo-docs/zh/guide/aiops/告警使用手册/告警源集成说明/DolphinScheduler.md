# DolphinScheduler
> DolphinScheduler 是一个分布式的易扩展的可视化数据工作流调度系统，旨在解决复杂的任务依赖管理与任务调度问题。它是由Apache孵化并成为顶级项目的一个开源项目，主要用于大数据领域中复杂的数据处理任务的调度和自动化。
>
